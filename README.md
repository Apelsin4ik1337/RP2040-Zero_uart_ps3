# Использование
 - Покдлючаете rp2040-zero к ПК и перекидываете файл из релизов, на появившийся накопитель.
 - SC_TX покдлючается к GP0, SC_RX подключаете к GP1, а также не забываем подключить GND.                    
   ```НЕ ПОДКЛЮЧАЕТЕ 3.3В ОТ КОНСОЛИ```
 - Покдлючите rp2040-zero к ПК, далее запустите любую программу умеющую работать с com портами, далее делайте что вам угодно.
   ```https://github.com/db260179/ps3syscon/blob/master/Linux/ps3_syscon_uart_script.py``` - это скрипт, при помощи него вы сможете вывести подробные логи с консоли.
